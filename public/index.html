Portfolio API

This is the API for the portfolio website. Here are the available endpoints:

Test API Connection

GET /api/test

A simple test endpoint to verify the API is working.

Get All Blogs

GET /api/blogs

Retrieve all blog posts.

Get Blog by ID

GET /api/blogs/:id

Retrieve a specific blog post by its ID.

Get All Projects

GET /api/projects

Retrieve all projects.

Get Featured Projects

GET /api/projects/featured

Retrieve all featured projects.

Get Project by ID

GET /api/projects/:projectId

Retrieve a specific project by its ID.
